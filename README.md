Women Safety Progressive Web App (PWA)

📌 Project Overview

The Women Safety Progressive Web App (PWA) is a web-based safety solution designed to help women during emergency
situations. The application provides quick access to emergency features such as  location sharing,call emergency contacts
,find nearby police station and call police . Being a Progressive Web App, it can be installed on mobile devices and works seamlessly across different platforms.


🚀 Features
Live Location Sharing
Share real-time location with trusted contacts during emergencies.

Emergency Contacts Management
Add, update, and manage emergency contacts.

One-Tap Emergency Assistance
Quickly access emergency services with a single click.

nearby police station
find nearest police station from the current location 

Progressive Web App Support
Installable on mobile and desktop devices.

Fast loading and responsive design.
Offline support for essential features.

User-Friendly Interface
Simple and intuitive design for quick access during critical situations.

⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/women-safety-pwa.git
Navigate to the project directory:
cd women-safety-pwa
Open the project in a browser or run it using a local server:
npx serve
Install the app on your device using the browser's "Add to Home Screen" option.
